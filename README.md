## Project Overview
This project performs exploratory data analysis on unemployment trends in Sri Lanka.
The analysis focuses on:
- Overall unemployment trends
- Gender-wise unemployment differences
- Relationship between inflation and unemployment

## Dataset
- Source: Sri Lanka Department of Census & Statistics (Labour Force Survey)
- Period: 2001â€“2024
- Variables:
  - Overall Unemployment

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Unemployment shows a long-term decline with spikes during economic crises.
- Female unemployment is consistently higher than male unemployment.
- Inflation and unemployment have a very weak correlation.
- Labour market reacts slowly to inflationary shocks.
